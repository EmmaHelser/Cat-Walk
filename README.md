## Dependencies

- React
- React-DOM
- Axios
- Babel
- Webpack
- Express.JS
- Node.JS
- Jest
- Chai

## Installation

To set up developer environment.

```
npm install
```

## Running

To configure Webpack.

```
npm run react-dev
```

To run server at port 3001.

```
npm run live-server
```
